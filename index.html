Présentation de l'exercice

Le Cyber-Humanum Est est un exercice de cyberdéfense qui a pour objectif de former les étudiants de l'université de Lorraine à la lutte informatique. Il est organisé par le COMCYBER (Commandement de la Cyberdéfense) et se déroule sur 3 jours.

LIO

La Lutte Informatique Offensive regroupe toutes les actions menées sur les infrastructures ennemies. L'objectif peut être d'accéder à des informations, bloquer des infrastructures ou modifier leur fonctionnement à notre avantage.

LID

La Lutte Informatique Defensive regroupe toutes les actions menées sur les infrastructures alliées, dans le but de les protéger.

LII

La Lutte Informatique d'Influence consiste en des actions de communication qui s'appuyent sur la LID et la LIO afin de gagner l'opinion publique.
